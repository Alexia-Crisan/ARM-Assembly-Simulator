Architecture Overview

32-bit ARM-inspired ISA. 16 general-purpose registers (R0–R15). R13 = SP (stack pointer), R14 = LR (link register), R15 = PC (program counter).

Unified memory — single flat address space: instructions at 0x0000, data/stack at 0x0100 (default 512-byte total, configurable).

Condition flags: N (negative), Z (zero), C (carry), V (overflow). Updated only by CMP or when the S-bit is set.

Data Processing Instructions

Encoding: cond | 00 | I | opcode(4) | S | Rn(4) | Rd(4) | operand2(12)
Mnemonic Syntax Operation Encoding note
MOV MOV Rd, #imm / Rm Rd = imm or Rm opcode 1101
ADD ADD Rd, Rn, Rm/#imm
ADD Rd, Rm/#imm		 Rd = Rn + op2
Rd = Rd + op2		 opcode 0100
SUB SUB Rd, Rn, Rm/#imm
SUB Rd, Rm/#imm		 Rd = Rn − op2
Rd = Rd − op2		 opcode 0010
CMP CMP Rn, Rm/#imm Sets flags from Rn − op2. No result written. opcode 1010, S=1
AND AND Rd, Rn, Rm/#imm Rd = Rn & op2 opcode 0000
ORR ORR Rd, Rn, Rm/#imm Rd = Rn | op2 opcode 1100
EOR EOR Rd, Rn, Rm/#imm Rd = Rn ^ op2 opcode 0001
MVN MVN Rd, Rm/#imm Rd = ~op2 opcode 1111
Immediate encoding: 12-bit field = rot(4) | imm8(8). Value = ROR(imm8, rot×2). Values must fit this form — not all 32-bit constants are encodable.

Multiply / Divide

Encoding: cond | 000000 | A | S | Rd(4) | 0000 | Rs(4) | 1001 | Rm(4)
Mnemonic Syntax Operation
MUL MUL Rd, Rm, Rs
MUL Rd, Rs		 Rd = Rm × Rs (integer, 32-bit result)
DIV DIV Rd, Rm, Rs
DIV Rd, Rs		 Rd = Rm ÷ Rs (integer division, truncated)
DIV by zero raises a runtime ZeroDivisionError and halts the simulation.

Load / Store

Encoding: cond | 01 | I | P | U | B | W | L | Rn(4) | Rd(4) | offset12(12)
Mnemonic Syntax Operation
LDR LDR Rd, [Rn]
LDR Rd, [Rn, #offset]		 Rd = mem[Rn + offset]
STR STR Rd, [Rn]
STR Rd, [Rn, #offset]		 mem[Rn + offset] = Rd
Word-aligned 32-bit transfers only. Access outside memory bounds raises MemoryError.

Stack Operations

Encoding: cond | 100 | P | U | S | W | L | Rn(4) | reg_list(16) — Base register = R13 (SP)
Mnemonic Syntax Operation
PSH / PUSH PSH {R0, R1, ...} SP -= 4 per register, writes to memory. Full-descending stack.
POP POP {R0, R1, ...} Reads from memory, SP += 4 per register.
SP (R13) starts at the top of the data region (empty-descending). First push writes to the last word of the data region.

Branch Instructions

Encoding: cond(4) | 101 | L | offset24(24) — Target = PC + 8 + offset×4
Mnemonic Condition Flags tested Notes
B Always — Unconditional jump
BEQ Equal Z=1
BNE Not equal Z=0
BLT Less than N≠V Signed
BGT Greater than Z=0 and N=V Signed
BGE Greater or equal N=V Signed
BLE Less or equal Z=1 or N≠V Signed
JMS Always (with link) — LR = PC+4, then jump. Use RET to return.
RET — — MOV PC, LR — returns from subroutine
ARM pipeline offset: when a branch executes, PC has already been fetched 8 bytes ahead, so target = PC + 8 + (offset × 4).

System Instructions

Encoding: 1111 | opcode(3) | operand(25) — identified by top nibble 0xF
Mnemonic Syntax Operation
HLT HLT Stop execution. Always place at end of program.
INP INP Rd Read integer from stdin → Rd (desktop mode only)
OUT OUT {R0, R1, ...} Print values of listed registers to stdout

Pseudo-Instructions

Assembler-only: pseudo-instructions are expanded at assemble time into real machine words. They use R12 as a scratch register (no PUSH/POP emitted — zero memory traffic).
Mnemonic Syntax Expands to Operation
INC INC Rd ADD Rd, Rd, #1 Rd++
DEC DEC Rd SUB Rd, Rd, #1 Rd--
CLR CLR Rd MOV Rd, #0 Rd = 0
LSL LSL Rd, Rn, #n
LSL Rd, #n		 MOV R12, #(1<<n)
MUL Rd, Rn, R12		 Logical shift left — multiply by 2ⁿ
LSR LSR Rd, Rn, #n
LSR Rd, #n		 MOV R12, #(1<<n)
DIV Rd, Rn, R12		 Logical shift right — integer divide by 2ⁿ
MOD MOD Rd, Rn, Rm DIV R12, Rn, Rm
MUL R12, R12, Rm
SUB Rd, Rn, R12 		Rd = Rn mod Rm
SWAP / SWP SWAP Rn, Rm MOV R12, Rn
MOV Rn, Rm
MOV Rm, R12		 Exchange two registers
LOOP LOOP label SUB R12, R12, #1
CMP R12, #0
BNE label 		Counted loop — R12 is the counter (set it before the loop)

Condition Flags

Flag Name Set when…
N Negative Result bit 31 = 1 (i.e. result is negative in signed interpretation)
Z Zero Result == 0
C Carry ADD: unsigned overflow; SUB/CMP: no borrow (rn ≥ val2)
V Overflow Signed overflow occurred
S-bit rule: Flags are only updated when the S bit (bit 20) is set in the encoding. CMP always sets S=1. Plain ADD/SUB/etc. have S=0 and leave flags unchanged.
