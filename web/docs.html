ARM Assembly Simulator – Instruction Reference

This simulator supports a simplified ARM-like instruction set, including core instructions, memory operations, branching, stack manipulation, and pseudo-instructions.

Data Processing Instructions

MOV – Move
  • MOV Rd, #imm → Rd = imm
  • MOV Rd, Rn → Rd = Rn
ADD – Addition
  • ADD Rd, Rn, Rm → Rd = Rn + Rm
  • ADD Rd, Rn, #imm → Rd = Rn + imm
  • ADD Rd, Rm → Rd = Rd + Rm
SUB – Subtraction
  • SUB Rd, Rn, Rm → Rd = Rn - Rm
  • SUB Rd, Rn, #imm → Rd = Rn - imm
  • SUB Rd, Rm → Rd = Rd - Rm
CMP – Compare
  • CMP Rn, Rm
  • CMP Rn, #imm
  • Updates flags only (no result stored)
Logical Operations
  • AND Rd, Rn, Rm
  • ORR Rd, Rn, Rm
  • EOR Rd, Rn, Rm
  • MVN Rd, Rm → Rd = NOT Rm

Memory Instructions

LDR – Load
  • LDR Rd, [Rn]
  • LDR Rd, [Rn, #offset]
STR – Store
  • STR Rd, [Rn]
  • STR Rd, [Rn, #offset]

Stack Instructions

PUSH
  • PUSH {Rlist}
  • Decrements SP and stores registers
POP
  • POP {Rlist}
  • Loads registers and increments SP

Branch Instructions

  • B label → unconditional jump
  • BL label → jump + save return address (LR)
  • RET → return (PC = LR)
Conditional branches:
  • BEQ → if Z == 1
  • BNE → if Z == 0
  • BGT → if Z == 0 and N == V
  • BLT → if N != V
  • BGE → if N == V
  • BLE → if Z == 1 or N != V

Multiply / Divide

  • MUL Rd, Rm, Rs → Rd = Rm * Rs
  • DIV Rd, Rm, Rs → Rd = Rm / Rs

System Instructions

  • HLT → stop execution
  • INP Rd → read input into register
  • OUT {Rlist} → print registers

Pseudo-Instructions

Simple:
  • INC Rd → ADD Rd, Rd, #1
  • DEC Rd → SUB Rd, Rd, #1
  • CLR Rd → MOV Rd, #0
Shift operations:
  • LSL Rd, Rn, #imm → Rd = Rn × (2^imm)
  • LSR Rd, Rn, #imm → Rd = Rn / (2^imm)
Advanced:
  • MOD Rd, Rn, Rm → Rd = Rn % Rm
  • SWAP Rn, Rm → swap values
  • LOOP label → counted loop using R12
